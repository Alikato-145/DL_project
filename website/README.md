# วิธีเปิดใช้งานตัว web app

### 1. เปิด terminal ขึ้นมา
### 2. ทำการติดตั้ง dependencies ด้วยคำสั่ง
```bash
pip install -r requirements.txt
 ```

### 3. cd เข้าไปที่โฟลเดอร์ website
```bash
cd website
```

### 4. ทำการรันคำสั่ง
```bash
streamlit run ./website/app.py
```
